# 🧠 Assessment Vocacional — Inteligencias Múltiples

Herramienta de orientación académica para estudiantes de secundaria basada en la **Teoría de las Inteligencias Múltiples de Howard Gardner**.

## ¿Qué es?

Un formulario interactivo de 40 preguntas que evalúa las 8 inteligencias del estudiante y genera un perfil vocacional completo con visualizaciones y recomendaciones de carreras.

## Inteligencias evaluadas

| Inteligencia | Descripción |
|---|---|
| 📚 Lingüística | Manejo del lenguaje oral y escrito |
| 🔢 Lógico-Matemática | Razonamiento lógico y numérico |
| 🎨 Espacial | Visualización y pensamiento en 3D |
| 🎵 Musical | Sensibilidad al ritmo y melodía |
| 🏃 Corporal-Kinestésica | Coordinación y habilidad física |
| 🌿 Naturalista | Comprensión del mundo natural |
| 🤝 Interpersonal | Empatía y relaciones sociales |
| 🧘 Intrapersonal | Autoconocimiento y reflexión |

## Características

- ✅ 40 preguntas de selección múltiple mutuamente excluyente
- 📊 Gráfico de barras animado por inteligencia
- 🕸 Mapa radar interactivo
- ⚡ Identificación de fortalezas y áreas de mejora
- 🎯 Recomendación de carreras alineadas al perfil
- 📄 **Exportación de resultados a PDF** (2 páginas, diseño profesional)

## Uso

Abre `index.html` en cualquier navegador moderno. No requiere instalación ni servidor.

## Tecnologías

- HTML5 + CSS3 + JavaScript vanilla
- [jsPDF](https://github.com/parallax/jsPDF) para exportación PDF
- Canvas API para el radar chart

## Basado en

> Gardner, H. (1983). *Frames of Mind: The Theory of Multiple Intelligences*. Basic Books.
